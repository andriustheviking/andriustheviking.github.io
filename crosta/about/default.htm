Hello!

Welcome to my Web Design I Final Project. The purpose of this project was to create a mobile-first, responsive splash page to fulfill the web objectives of a (fictional) organization using bootstrap's responsive template. The project included:

  1. Strategy Document

    The Strategy Document outlines the objectives of the organization, how the website can meet those objectives, and the tools used to do so.

    You can view a pdf copy of the strategy document here.

  2. Wireframe

    For this website, I drew two separate wireframes, illustrating how the elements from a small, mobile format would translate into a large, desktop format.

    For the mobile wireframe, I stacked the elements on top of each other with hire priority elements on top
    I then built a larger wireframe, showing how those elements would expand to fill a desktop-sized screen. Overlayed on the larger wireframe are the basic row and column breakdowns for the bootstrap template. During development, I chose to use three main columns with images in the middle, instead of two.

    You can view the mobile wireframe here and the desktop wireframe here.

  3. Persona Workflows

    Three User Personas were required in with the strategy of the website's design. These personas were used to keep the website's functionality targeted to its different types of users. Included with the Persona document are three videos showing how the website helps its users accomplish their goals. Including one instructional video to as a training video on the website's use.

    You can view a pdf copy of the User Persona document here.
